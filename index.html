Stack

Stack is a linear data structure which follows a particular order in which the operations are performed. The order may be LIFO(Last In First Out) or FILO(First In Last Out).

There are many real-life examples of a stack. Consider an example of plates stacked over one another in the canteen. The plate which is at the top is the first one to be removed, i.e. the plate which has been placed at the bottommost position remains in the stack for the longest period of time. So, it can be simply seen to follow LIFO(Last In First Out)/FILO(First In Last Out) order.

The push function first checks whether the stack is full or empty with the help of isFull() function. If it is not full, it increments the top counter and adds the element to that

The push function checks whether the stack is empty or not with the help of isEmpty() function.If it is not empty, then it decrements the top pointer which removes (pops) the element in the previous address.
